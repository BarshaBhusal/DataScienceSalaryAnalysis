# SalaryAnalysis
### Overview

This project analyzes a dataset of Data Science salaries to understand factors that affect earnings and identify trends in the field. Python is used to clean, process, and explore the data, revealing insights about experience, job role, company size, location, and remote work.

### Dataset

The dataset contains columns like:

work_year, experience_level, employment_type, job_title

salary_in_usd, employee_residence, remote_ratio

company_location, company_size

### Data Preparation

* Loaded the data into a pandas DataFrame

* Removed unnecessary columns (salary, salary_currency)

* Dropped missing values

* Checked for unique values and cleaned the dataset

### Analysis

* Explored trends in salaries by experience, role, location, and company size

* Used summary statistics and visualizations to find patterns

* Highlighted key factors affecting Data Science salaries
